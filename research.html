Research

Exploring the frontiers of AI and machine learning

Research Areas

Reinforcement Learning

Developing and applying deep RL methods like PPO and SAC to real-world tasks with continuous control. My work focuses on improving sample efficiency and stability in complex environments.

Key Topics:

  • Proximal Policy Optimization (PPO)
  • Soft Actor-Critic (SAC)
  • Deep Q-Networks (DQN)
  • Multi-agent Reinforcement Learning

Evolutionary Computing

Using genetic algorithms and particle swarm optimization to explore search space and optimize performance. I investigate hybrid approaches combining evolutionary methods with machine learning.

Key Topics:

  • Genetic Algorithms (GA)
  • Particle Swarm Optimization (PSO)
  • Simulated Annealing (SA)
  • Evolutionary Strategies

Machine Learning

Building predictive models and analyzing data using supervised and unsupervised learning techniques. My research includes deep learning applications and model interpretability.

Key Topics:

  • Deep Neural Networks
  • Supervised Learning
  • Unsupervised Learning
  • Model Interpretability

Intelligent Optimization

Combining learning with search strategies to solve complex optimization problems in dynamic environments. This includes meta-heuristic approaches and adaptive algorithms.

Key Topics:

  • Meta-heuristic Algorithms
  • Adaptive Optimization
  • Multi-objective Optimization
  • Dynamic Optimization

Current Research Projects

Multi-Agent Reinforcement Learning for Resource Allocation

Ongoing

Developing a novel multi-agent reinforcement learning framework for dynamic resource allocation in distributed systems. This project aims to improve efficiency and fairness in resource distribution across multiple agents.

Duration: 2023 - Present
Technologies: Python, PyTorch, Gym, Multi-Agent Systems
Status: Data collection and initial model development

Evolutionary Algorithms for Neural Architecture Search

Planning

Investigating the use of evolutionary algorithms to automatically design neural network architectures. This research focuses on reducing computational cost while maintaining model performance.

Duration: 2024 - 2025
Technologies: Python, TensorFlow, Evolutionary Computation
Status: Literature review and methodology design
